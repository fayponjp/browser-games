# GameTryal 

This project is a living and evolving playground for developing browser games.

Currently, two games are available:

- Not-Wordle, a game developed from the NY Times' popular [Wordle](https://www.nytimes.com/games/wordle/index.html) game. Infinitely repeatable.
- Poke-Hangman, a twist on the classic hangman. Data sourced from the [poke-api project](https://pokeapi.co/).
